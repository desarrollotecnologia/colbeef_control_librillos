Compartir: enlace, iframe y archivos

Enlace directo (misma base que abre usted ahora). Péguelo en el otro sistema como destino de un botón o envíelo por correo/chat.

Iframe desde otro portal: use la misma URL en src solo si ambos sitios son el mismo origen o tiene proxy/CORS resuelto; si no, abra siempre en nueva pestaña.

Paquete de archivos (gráficos sin CDN): en el servidor deben existir juntos dashboard-ejecutivo-librillos.html y la carpeta vendor/chart.umd.min.js (ya incluida en este proyecto). No hace falta conexión a Internet para dibujar los gráficos.

Los datos siguen saliendo de la API de este programa (/api/librillos/…), así que quien abra el enlace necesita acceso de red al mismo host donde corre Colbeef control librillos.
